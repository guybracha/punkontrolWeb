# עדכונים חדשים באתר Punkontrol

## ✨ תכונות חדשות שנוספו

### 1. 📝 עריכת פרטי יצירות ופוסטים

עכשיו ניתן לערוך את הפרטים של יצירות ופוסטים שהעלית!

**איך להשתמש:**
- גש ליצירה או לפוסט שלך
- לחץ על כפתור העריכה ✏️ שמופיע בפינה השמאלית העליונה
- ערוך את הכותרת, תיאור ותגיות
- לחץ "שמור שינויים"

**הערות:**
- רק בעלי היצירה/פוסט יכולים לערוך אותם
- לא ניתן לשנות את התמונה (רק את הטקסט)
- השינויים נשמרים מיידית במסד הנתונים

---

### 2. 🎥 הטמעת סרטוני YouTube בפוסטים

כעת ניתן להוסיף סרטוני YouTube לפוסטים!

**איך להשתמש:**
1. כשאתה יוצר פוסט חדש, תראה שדה חדש: "סרטון YouTube (אופציונלי)"
2. העתק את הקישור המלא של הסרטון מ-YouTube
   - למשל: `https://www.youtube.com/watch?v=dQw4w9WgXcQ`
   - או: `https://youtu.be/dQw4w9WgXcQ`
3. הדבק את הקישור בשדה
4. פרסם את הפוסט

**תוצאה:**
- הסרטון יוטמע באופן אוטומטי בפוסט
- המבקרים יוכלו לצפות בסרטון ישירות מהאתר
- הסרטון מוצג בפורמט 16:9 מתאים

**דוגמאות לפוסטים עם סרטונים:**
- מדריכי אמנות
- תהליך יצירה (speedpaint/timelapse)
- הרצאות ושיחות
- מוזיקה וסאונד ארט

---

### 3. 📊 תיקון ספירת יצירות ופוסטים

תוקן הבאג שגרם לספירות להציג 0 גם כשהיו יצירות/פוסטים!

**מה תוקן:**
- כעת כשמעלים יצירה חדשה, הספירה מתעדכנת אוטומטית
- כשמעלים פוסט חדש, הספירה מתעדכנת אוטומטית
- בפרופיל מוצג גם מספר היצירות וגם מספר הפוסטים

**איך לתקן ספירות קיימות:**
1. היכנס לפרופיל שלך
2. לחץ על הכפתור "🔧 תקן ספירות"
3. המערכת תספור אוטומטית את כל היצירות והפוסטים שלך
4. תופיע הודעה עם המספרים המעודכנים

**הערות:**
- כל משתמש יכול לתקן את הספירות שלו בלבד
- הפעולה בטוחה לחלוטין ואינה מוחקת שום תוכן
- מומלץ להריץ את זה פעם אחת אם היו לך תכנים שהועלו לפני העדכון

---

## 🔧 קבצים שהשתנו

1. **src/routes/Artwork.jsx** - הוספת אפשרות עריכה ליצירות
2. **src/routes/Post.jsx** - הוספת עריכה ותמיכה ב-YouTube
3. **src/routes/Upload.jsx** - שדה YouTube ועדכון ספירות אוטומטי
4. **src/routes/Profile.jsx** - כפתור תיקון ספירות והצגת מספר פוסטים
5. **src/services/posts.api.js** - תמיכה ב-YouTube ועדכון פונקציית createPost
6. **src/lib/fixCounts.js** - פונקציות חדשות לתיקון ספירות

## 📱 שימוש באתר

### עריכת יצירה:
```
1. גש ל-/art/[your-artwork-slug]
2. לחץ ✏️
3. ערוך ← שמור
```

### עריכת פוסט:
```
1. גש ל-/post/[post-id]
2. לחץ ✏️
3. ערוך ← שמור
```

### הוספת YouTube לפוסט:
```
1. גש ל-/upload
2. בחר "פוסט"
3. מלא את הפרטים
4. הדבק קישור YouTube
5. פרסם
```

### תיקון ספירות:
```
1. גש לפרופיל שלך: /u/[username]
2. לחץ "🔧 תקן ספירות"
3. ✓ סיום!
```

## 💡 טיפים

1. **תגיות:** השתמש בתגיות עקביות כדי שיהיה קל למצוא את התוכן שלך

3. **עריכה:** שמור את הטקסט המקורי במקום אחר לפני שאתה עורך, למקרה שתרצה לחזור אליו
4. **ספירות:** אם הספירות לא נראות נכון, פשוט השתמש בכפתור התיקון - זה תמיד עובד!
